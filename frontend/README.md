# HealthConnect - Healthcare Web Application

A modern, professional healthcare platform built with React + Vite, featuring video consultations, appointment booking, and health tracking.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📋 Features

- **Home Page**: Hero section with features and call-to-action
- **Appointment Booking**: Browse and book appointments with doctors
- **Video Consultations**: Secure video calls with healthcare professionals
- **Health Dashboard**: Track vital statistics and medical history
- **Contact Form**: Get in touch with support team

## 🎨 Design

- Modern blue and white theme
- Responsive design for all devices
- Smooth animations and transitions
- Professional UI/UX

## 🛠️ Tech Stack

- React 18
- Vite
- CSS3 with custom design system
- Google Fonts (Inter)

## 📱 Pages

1. **Home** - Landing page with hero and features
2. **Video Call** - Video consultation interface
3. **Appointment** - Doctor browsing and booking
4. **My Stats** - Health metrics dashboard
5. **Contact Us** - Contact form and information

## 🌐 Live Development Server

The app runs on `http://localhost:5173` when you start the dev server.
